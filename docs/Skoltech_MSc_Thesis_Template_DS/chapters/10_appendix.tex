\chapter*{Appendix: editing prompts}
\addcontentsline{toc}{chapter}{Appendix: editing prompts}

The appendix lists the source, target, and segmentation prompt triplets used in every editing task referenced in the thesis. All targets retain the source's scene description verbatim and change only the object noun; the segmentation prompt names the object that SAM~3.1 should locate.
